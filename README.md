# Spread_Arbitrage_Tester

Проект предназначен для тестирования стратегии арбитража основываясь на исторических данных с бирж OKX и Bybit. 
Система позволяет:
- автоматически загружать исторические данные через API,
- синхронизировать их по времени,
- вычислять разницу (spread) между ценами,
- симулировать арбитражные сделки,
- рассчитывать ключевые метрики: PnL, drawdown, количество сделок и прибыль на сделку.

Проект написан на C++ (основной движок) с использованием Python для загрузки данных.
